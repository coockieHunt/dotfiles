# 🌿 dotfiles — my Hyprland config

> **EN** | [FR](#-dotfiles--ma-config-hyprland)

## About

This is my personal Hyprland config. I'm sharing it in case it's useful to someone, but keep in mind it's built around my own setup so it probably won't work perfectly out of the box on yours.

Feel free to take whatever you find useful, adapt it, or use it as inspiration. That's kind of the whole point of sharing dotfiles. 🙂

## 🛠️ What's in here

`.config/`
- `eww/` — widgets
- `fuzzel/` — app launcher
- `hypr/` — Hyprland config
- `swaync/` — notification center
- `swayosd/` — on-screen display (volume, brightness)
- `waybar/` — status bar
- `wofi/` — app launcher

## 🚀 Getting started

```bash
git clone https://github.com/coockieHunt/dotfiles
```

Then symlink or copy the parts you want. I'd recommend going through the configs before applying anything wholesale some paths and monitor settings are specific to my machine.

## 💻 My setup

- **WM**: Hyprland
- **Distro**: Arch-based
- **Monitors**: hardcoded in `hypr/`, you'll want to update the `monitor=` lines
- **Machine**: thinkpad T14 GEN2 hd screen
